### 💼 Sobre mim

Desenvolvedor Full Stack e bacharel em Ciência da Computação, com sólida base em engenharia de software e arquitetura de sistemas. Atualmente, atuo como Desenvolvedor Autônomo construindo soluções end-to-end para clientes nacionais e internacionais.

Sou especialista no ecossistema Front-end com **React, Next.js e Angular**, focado na construção de interfaces de alta performance e arquiteturas escaláveis (Microfrontends e Monólitos Modulares). No Back-end, minhas tecnologias de eleição para desenvolvimento de APIs são **Node.js (Fastify)** — compondo minha stack principal ao lado do React — e **Java (Spring Boot)** — formando uma estrutura robusta em conjunto com o Angular. Além da minha stack principal, possuo conhecimentos complementares em ferramentas como PHP (Laravel) e Express, aplicando boas práticas de design de software em todo o ciclo de vida da aplicação.
